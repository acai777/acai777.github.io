An open-source, self-hosted framework which updates caches in near real-time.

Read the Case Study

Willow applies Change Data Capture to monitors changes in a user's database and reflects row-level changes in a user's cache. Without requiring in-depth technical knowledge and expertise, data pipelines connecting databases and caches can be created or deleted using Willow's web user interface, simplifying setup and teardown.

Willow is built with a variety of tools, including Debezium, Apache Kafka, Docker, TypeScript, and PostgreSQL.
